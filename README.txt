It should be noted that this application is currently quite vulnerable to SQL-injection attacks; proceed with caution.

Please see "requirements.txt" for packages used in the creation of this application.
Not every line of code in this program is homebrewed :)

The application is intended to be run from the command line (ideally using a virtual environment).
To do this; perform the following from the command line:
-> Navigate to the directory of MyQueryHouse/
-> Execute: "python3 -m venv env"
-> Execute: "source env/bin/activate"
-> Execute: "pip3 install -r requirements.txt"
-> Execute: "python3 app.py"
